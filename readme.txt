Social Icons Widget to displays links to social sharing websites. Currently its Supports Only 15 sites.

== Description ==

> Did you find this plugin helpful? Please consider leaving a 5-star review for more features.

These Plugin helps you to add quickly icons with links to your profiles from different social networks. 


= Features: =

* 15 Sites/Social Networks supported
* ShortCode Integration
* FontAwesome Integration

= ShortCode =

You can use 2-shortcode .

1. `[cdx_social_share]` :- this shortcode can display share buttons on posts & pages .

2. ```[cdxSocialLinks fb="facebook_url" tw="twitter_url" yt="youtube_url" ins="instagram_url" lkin="linkedIn_url" email="email_address" be="behance_url" dr="dribbble_url" gt="github_url" pin="pinterest_url" tum="tumblr_url" sound="soundcloud_url" cpen="codepen_url"]```

This second shortcode can display some social buttons . Attributes are :-

* **fb (facebook)**
* **tw (twitter)**
* **ins (Instagram)**
* **lkin (LinkedIn)**
* **email (Email)**
* **be (Behance)**
* **dr (dribbble)**
* **gt (Github)**
* **pin (Pinterest)**
* **tum (tumblr)**
* **sound (SoundCloud)**
* **cpen (codepen)**

> This attributes are not required . if you use 2 or 3 , Display only these 2 or 3 , others will be hide.

= Icons =

Icons are provided by [fontawesome](https://fontawesome.com) icon font.

Twitter, Facebook, Google+, Instagram, Pinterest, LinkedIn, SoundCloud, YouTube, Tumblr, Reddit, Dribbble, Behance, GitHub.

== Installation ==

Simply search for the plugin via the **Plugins -> Add New** dialog and click install, or download and extract the plugin, and copy the plugin folder into your wp-content/plugins directory and activate.

After that go to the **Widgets** page and add the widget **Social Links** to a widget area like Sidebar.
